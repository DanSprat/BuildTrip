# BuildTrip

Flutter-приложение для планирования поездок.

## Что уже реализовано

- Главный экран со списком поездок.
- Экран выбранной поездки со списком дней.
- Экран дня с подробностями и списком мест для посещения.

## Как запустить

1. Установить Flutter SDK и добавить `flutter` в `PATH`.
2. В папке проекта выполнить:

```bash
flutter create .
flutter pub get
flutter run
```

Команда `flutter create .` создаст платформенные папки для Android и iOS.
